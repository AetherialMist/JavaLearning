# Challenges

Create new projects for each of the following challenges:

1. Print three different messages.
   * _Hint: Add more print statements on new lines below the first._
2. Print five different messages.
3. Print a pattern:
   ```text
      J    A    V     V    A
      J   A A    V   V    A A
   J  J  AAAAA    V V    AAAAA
    JJ  A     A    V    A     A
   ```
